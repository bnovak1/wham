# wham

Tested on Version 2.0.9.

Download WHAM from [here](http://membrane.urmc.rochester.edu/?page_id=126).

WHAM documentation can be found [here](http://membrane.urmc.rochester.edu/sites/default/files/wham/doc.pdf).
